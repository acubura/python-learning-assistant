## Overview:
This interactive assistant guides beginners through Python's core concepts using a friendly, conversational approach. Instead of traditional textbook explanations, it uses real-world analogies and step-by-step guidance to make programming concepts intuitive and memorable.

## Features:
It has following features:
- 8 Core Topics: Variables, functions, operators, input handling, and more
- Beginner-Friendly: No prior programming experience required
- Interactive Learning: Choose topics and learn at your own pace
- Real-World Analogies: Complex concepts explained simply

## Quick Start:
To quickly install and use it, use this following method:

git clone https://github.com/acubura/python-learning-ai-assistant.git
- cd python-learning-ai-assistant
- python ai_assistant.py

## Topics Covered
Hello World - Your first Python program

1. Functions - Reusable code blocks
2. Variables - Data storage and types
3. Relational Operators - Comparison tools
4. Assignment Operators - Value assignment shortcuts
5. Logical Operators - Decision-making
6. Type Conversion - Changing data types
7. Input Function - User interaction

## Requirements
This project has following requirements:
- Python 3.x or latest version.
- No external dependencies
